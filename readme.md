#README

This is an ascii-based computer architecture ide.

Syntax:
command|picture
-|-
bus|║ ═ ╠ ╩ ╦ ╣ ╚ ╗ ╔ ╝ ╬
wire|│ ─ ├ ┴ ┬ ┤ └ ┐ ┌ ┘ ┼
disconnect|# (to differ between connected wires and crossing wires)
diode|^ v < >
logic gates|* + x !

```
a
╠═══╦════╦════╦════╗
0   1┌┐  2┌┐  3┌┐  4
│   ├┘v  ├┘v  ├┘v  │
├─┬─┼>*┬─┼>*┬─┼>*┐ │
v │ v  │ v  │ v  │ v
! └>x  └>x  └>x  └>x
0   1    2    3    4
╠═══╩════╩════╩════╝
b
```
